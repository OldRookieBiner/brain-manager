## 📖 使用示例

### 示例 1: 会话提炼
```bash
# 提炼架构设计会话
/knowledge_summarize --title "爬虫引擎架构" --category "架构"

# 输出：
# ✅ 已提炼知识卡片：爬虫引擎架构
# 📁 保存到：docs/knowledge-base/爬虫引擎架构.md
# 📒 同步到：思源笔记/OpenClaw 知识库/架构/爬虫引擎架构
```

### 示例 2: 知识检索
```bash
# 检索 aiohttp 相关知识
/knowledge_search --query "aiohttp 异步请求" --limit 3

# 输出：
# 🔍 已检索到 3 篇相关笔记：
# 1. 爬虫引擎架构 - 相关性 85%
# 2. 技术选型决策 - 相关性 72%
# 3. 并发连接数问题 - 相关性 68%
```

### 示例 3: 知识管理
```bash
# 列出所有架构文档
/knowledge_list --category "架构"

# 获取特定文档
/knowledge_get --id "爬虫引擎架构"
```
