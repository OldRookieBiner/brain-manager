- ✅ 配置远程仓库

### 步骤 2：在 GitHub 创建仓库

1. 访问：https://github.com/new
2. 填写：
   - **Repository name**: `openclaw-brain`
   - **Description**: `OpenClaw Brain Manager - 大脑管家`
   - **Visibility**: Public
   - ❌ 不要勾选任何初始化选项
3. 点击 "Create repository"

### 步骤 3：推送代码

```bash
git push -u origin main
```
