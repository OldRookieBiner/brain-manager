# OpenClaw Knowledge Skill

这是一个强大的知识管理插件，帮助 OpenClaw 自动整理对话内容，构建个人知识库。

## 核心功能

- **会话提炼**：自动将对话内容提炼为结构化知识卡片
- **思源同步**：将知识卡片同步到思源笔记指定笔记本
- **智能检索**：从本地知识库检索相关信息作为上下文
- **自动分类**：支持架构、模块、规范、问题等多种分类

## 使用场景

1. 完成一个功能模块开发后，自动提炼技术文档
2. 解决复杂问题后，记录解决方案和决策过程
3. 新话题开始时，检索既有知识库保持一致性
4. 定期整理会话历史，形成团队知识资产
